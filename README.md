## 📋 Projektübersicht

**Join-Public** ist eine Full-Stack-Webanwendung zur effizienten Verwaltung von Teamaufgaben. Die App implementiert ein Kanban-Board mit Drag & Drop, eine Kontaktverwaltung, eine Aufgaben-Erstellungsmaske sowie ein Dashboard mit Kennzahlen – alles in reinem HTML, CSS und JavaScript ohne externe Bibliotheken.

### Kernfunktionen

| Funktion | Beschreibung |
|----------|---------------|
| 🎯 **Kanban-Board** | Aufgaben nach Status (ToDo, In Progress, Done) mit Drag & Drop |
| 👥 **Kontaktverwaltung** | Zentrale Liste mit Teammitgliedern und Detailansicht |
| ➕ **Aufgaben-Editor** | Titel, Beschreibung, Fälligkeitsdatum, Priorität, Zuweisung |
| 📊 **Dashboard (Summary)** | KPIs, Aufgabenanzahl, bevorstehende Termine |
| ⚖️ **Rechtstexte** | Impressum und Datenschutzerklärung integriert |
| 📱 **Responsive Design** | Desktop-, Tablet- und Smartphone-optimiert |
| 💾 **Persistenz** | Browser-LocalStorage für clientseitige Datenspeicherung |
